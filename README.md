# Barcode PDF Generator

Утилита с современным веб-интерфейсом для массовой генерации PDF-файлов со штрих-кодами (Code128) из CSV-таблицы.
Формат генерируемых штрих-кодов: 58x40 мм (стандартный размер термоэтикетки для маркетплейсов).

## Возможности
- Чтение CSV файлов в различных кодировках (`utf-8`, `cp1251` и т.д.).
- Веб-интерфейс для предпросмотра данных перед генерацией.
- Пакетное скачивание всех штрих-кодов в одном ZIP-архиве.
- CLI-версия для автоматизации.

## Требования к CSV файлу
CSV файл должен содержать следующие колонки:
- **Артикул продавца** (используется как имя файла)
- **Баркоды** (цифры штрих-кода)
- **Наименование** (название товара, которое будет напечатано)
- *Опционально:* **Артикул WB** 

## Установка и Запуск

### Для Windows
Просто дважды кликните по файлу `start_win.bat`. 
Скрипт автоматически установит нужные библиотеки и откроет браузер.

### Для macOS / Linux
Дважды кликните по файлу `start_mac.command` или выполните в терминале:
```bash
./start_mac.command
```

### Запуск CLI версии (для разработчиков)
Вы можете использовать скрипт без графического интерфейса:
```bash
pip install -r requirements.txt
python generate_pdf.py Товары.csv barcodes_output/
```
